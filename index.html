RAMCleaner

RAMCleaner is a free program for Linux to clean out your sytem's RAM to improve performance

View project on GitHub

Welcome to RAMCleaner

RAMCleaner cleans out the unused cache in your system to impove performance and to allow programs to utilize it more. To use RAMCleaner, put the ramcleaner.sh file in your home folder and type

sudo sh ramcleaner.sh

If you do not use sudo or root, account the program will not run because it needs root access.
